# wxAutoIDE
实现微信小程序自动登陆、代码打包、带上上传功能

![][php]  ![][1] ![][2] 

## 项目依赖

### wxLogin  
[WXA-IDE/wxLogin][wxLogin]是用户对微信小程序IDE进行登陆的接口  

### wxPacker  
[WXA-IDE/wxPacker][wxPacker]是用户对微信小程序IDE进行打包的接口  

### wxUpload  
[WXA-IDE/wxUpload][wxUpload]是用户对微信小程序IDE进行上传的接口
